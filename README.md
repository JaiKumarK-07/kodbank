## Quick Start

1. Install dependencies:
```
npm install
```

2. Start the application:
```
npm start
```

3. Open your browser and navigate to:
```
http://localhost:3000
```

## Usage

1. Register a new account on the homepage
2. Login with your credentials
3. Click "Check Balance" to see your balance with celebration animation

## Technical Stack

- Backend: Node.js + Express
- Database: SQLite (sqlite3 - pure JavaScript)
- Authentication: JWT with bcryptjs (no native dependencies)
- Frontend: Vanilla HTML/CSS/JavaScript

No external database, build tools, or Visual Studio required - everything works out of the box!
